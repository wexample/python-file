# wexample-file

Package that allows you to manage the state of files and directories using YAML configuration files.

Version: 0.0.6

## Requirements

- Python >=3.10

## Dependencies

- wexample-helpers==0.0.67

## Installation

```bash
pip install wexample-file
```
